Class PluginInfoModel

插件信息模型

对应插件目录下 info.json

约定: 插件文件夹名=PluginID

约定: 插件文件夹名=插件主程序集(Assembly)名 .dll

eg: plugins/payment payment.dll

约定: 插件文件夹下 logo.png 为插件图标

约定: 插件文件夹下 README.md 为插件说明文件

约定: 插件文件夹下 settings.json 为插件设置文件
